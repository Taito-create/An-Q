# 🚀 An-QアプリをGitHub Pagesで公開する簡単な方法

## 📋 **ステップ1：GitHubリポジトリ作成**

1. **GitHubにログイン**
2. **「+」ボタン →「New repository」**
3. **Repository name：** `an-q-app`
4. **☑️ Public**を選択
5. **「Create repository」**

## 📁 **ステップ2：ファイルアップロード**

1. **「Add file」→「Upload files」**
2. **以下のファイルをドラッグ：**
   - `app.html` → `index.html`にリネーム
   - `public/`フォルダ全体
3. **「Commit changes」**

## ⚡ **ステップ3：GitHub Pages有効化**

1. **リポジトリの「Settings」**
2. **「Pages」セクション**
3. **Source：**「Deploy from a branch」
4. **Branch：**「main」
5. **「Save」**

## 🌐 **公開URL**

数分後：
```
https://[あなたのGitHubユーザー名].github.io/an-q-app/
```

## 📱 **PWAインストール方法**

### **iPhone**
1. Safariで上記URLを開く
2. 共有ボタン →「ホーム画面に追加」
3. 「An-Q」アプリをインストール

### **Android**
1. Chromeで上記URLを開く
2. メニュー（⋮）→「ホーム画面に追加」
3. 「An-Q」アプリをインストール
