# 🚀 GitHub PagesでAn-Qアプリを公開する方法

## 📋 **概要**
GitHub Pagesを使えば、無料で安定したホスティングが可能。ファイアウォール設定不要！

## 🛠️ **手順**

### **ステップ1：GitHubリポジトリ作成**
1. GitHubにログイン
2. 「New repository」をクリック
3. リポジトリ名：`an-q-app`
4. Publicを選択
5. 「Create repository」をクリック

### **ステップ2：ファイルをアップロード**
1. 「Add file」→「Upload files」をクリック
2. 以下のファイルをドラッグ＆ドロップ：
   - `app.html`
   - `public/` フォルダ全体
3. 「Commit changes」をクリック

### **ステップ3：GitHub Pagesを有効化**
1. リポジトリの「Settings」をクリック
2. 「Pages」セクションまでスクロール
3. Source：「Deploy from a branch」を選択
4. Branch：「main」を選択
5. 「Save」をクリック

### **ステップ4：公開URLを確認**
数分後、以下のURLでアクセス可能：
```
https://[ユーザー名].github.io/an-q-app/
```

## 📱 **PWAインストール手順**

### **iPhoneユーザー**
1. Safariで上記URLにアクセス
2. 共有ボタン → 「ホーム画面に追加」
3. 「An-Q」アプリがインストールされる

### **Androidユーザー**
1. Chromeで上記URLにアクセス
2. メニュー（⋮）→ 「ホーム画面に追加」
3. 「An-Q」アプリがインストールされる

## ✅ **メリット**
- 🆓 完全無料
- 🌐 世界中からアクセス可能
- 📱 iPhone/Android両方で簡単インストール
- 🔧 ファイアウォール設定不要
- ⚡ 高速で安定

## 🔄 **更新方法**
アプリを更新する場合：
1. ファイルを修正
2. GitHubに再度アップロード
3. 数分で自動更新される
